# Orientação a Objetos e UML: Diagramação de Classes do iPhone

![terraform](https://img.shields.io/badge/-UML-white?style=for-the-badge&logo=UML&color=FABD14&logoColor=white)

Este é um diagrama UML que representa a estrutura de classes e interfaces para um sistema que inclui a modelagem do iPhone, um dispositivo que incorpora funcionalidades de um reprodutor de música, um telefone e um navegador de internet. Este diagrama foi criado como parte do desafio de projeto do **Santander Bootcamp 2024 - Java** na DIO.me.

![iphone](https://github.com/gaarciia/Desafio-Dio-Java/blob/master/UML/Diagrama%20ER%20de%20banco%20de%20dados%20IPHONE%202007.png?raw=true)
